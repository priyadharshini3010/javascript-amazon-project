# Amazon Clone Project

An Amazon-style e-commerce website built using HTML, CSS, JavaScript (frontend) and Node.js/Express (backend).

## Features
- Product listing page
- Shopping cart
- Checkout page
- Delivery options

## Technologies Used
- HTML, CSS, JavaScript (Frontend)
- Node.js, Express.js (Backend)
- JSON (Product data)

## How to Run
1. Clone the repository
2. Navigate to the backend folder: `cd backend`
3. Install dependencies: `npm install`
4. Start the server: `node server.js`
5. Open `amazon.html` using Live Server
